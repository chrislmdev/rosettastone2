Browser import (CSV / XLSX)

Drops or picks files into this app only (no server). Use the same header row as CSV templates. For .xlsx, only the first worksheet is read. Names like aws_pricing.csv or azure_exceptions.xlsx are required so CSP and type can be inferred.

CSV or XLSX (converted to CSV in the browser before upload). Filenames should include CSP and type (e.g. oci_pricing.xlsx, aws_parent_mapping.csv). Set Import month above, add files, then Process all — uploads, waits for processing, and reloads catalogs from the backend.
